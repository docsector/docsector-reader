## Demonstração

Veja como os títulos aparecem quando renderizados a partir do Markdown:

## Este é um H2

### Este é um H3

#### Este é um H4

##### Este é um H5

###### Este é um H6

Cada título acima cria uma âncora na árvore de ToC à direita. Clique em qualquer título para navegar diretamente até ele. Note como o hash da URL atualiza e a árvore de ToC destaca o título selecionado.

## Uso no Markdown

Simplesmente use a sintaxe padrão de heading Markdown nos seus arquivos de conteúdo `.md`:

```
## Seção Principal
Conteúdo aqui...

### Sub-Seção
Mais conteúdo...

#### Detalhe
Ainda mais conteúdo...
```

DPageSection automaticamente mapeia cada nível de heading ao componente DH correspondente.
